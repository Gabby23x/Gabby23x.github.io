MLS-MPM and CPIC (2025)

Image Space Modal Basis in JS and Python (2024)
Implements the 2015 paper "Image-Space Modal Bases for Plausible Manipulation of Objects in Video" by Davis et al., extending the original work with modal warping. Developed modal analysis pipeline in Python and custom JavaScript rasterizer for real-time rendering.
GitHub (Coming Soon)
Obstruction Removal - SliceSplatting (2024)
Modifies Gaussian Splatting to remove obstructions blocking the view of objects of interest in 3D scene reconstruction.

Automates generation of node-edge graph programs from hand-drawn sketches using vision-language models and differential evolution. CS 6172 Final project.
